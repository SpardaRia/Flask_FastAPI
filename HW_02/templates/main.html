Задание

Создать страницу, на которой будет форма для ввода имени и электронной почты, при отправке которой будет создан cookie файл с данными пользователя. Также будет произведено перенаправление на страницу приветствия, где будет отображаться имя пользователя.
На странице приветствия должна быть кнопка "Выйти", при нажатии на которую будет удален cookie файл с данными пользователя и произведено перенаправление на страницу ввода имени и электронной почты.
